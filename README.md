# Task Near

Task Near is a service-based application inspired by UrbanClap. 
It helps users find and book nearby service providers for daily needs like carpentry, electrician, welding, painting, and other home services using location-based technology.

## Features
- Search nearby service providers
- Book service instantly
- Location-based service request

## Tech Stack
- HTML
- CSS
- JavaScript

## How to use
Open the index.html file in your browser.
